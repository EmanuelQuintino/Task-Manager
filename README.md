# Task Manager

![home-layout](./src/assets/login-layout.png)

The Task Manager is a `Full-Stack` Web Application developed as a final project of `Reprograma Juc√°s` [Web Development Course](https://emanuelquintino.github.io/Page-WDC/). This application allows the user to organize their tasks, offering management and monitoring capabilities for their activities.

## Features

- Integration with API
- Listing and pagination with all Tasks
- Search field when typing filter
- Application Deployment

## Technologies

- `React`
- `TypeScript`
- `Style Components`
- `React Router DOM`
- `React Hook Form`
- `React Tanstack Query`
- `Axios`

## Project Configuration

```bash
# clone the repository
$ git clone https://github.com/EmanuelQuintino/Task-Manager.git

# install dependencies
$ npm install

# run the project
$ npm run dev
```
